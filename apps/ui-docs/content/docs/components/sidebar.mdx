## API Reference

This component set is implemented in `@kaotypr/ui` and uses Base UI utilities like `mergeProps` and `useRender` to support render composition on selected sub-components.

### SidebarProvider

Context provider that controls sidebar open/collapsed state, supports controlled/uncontrolled usage, persists state via a cookie, and adds a keyboard shortcut (Cmd/Ctrl+B).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultOpen` | `boolean` | `true` | Initial open state (uncontrolled) |
| `open` | `boolean` | - | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | - | Called when open state changes |
| `className` | `string` | - | Additional CSS class names |
| `style` | `React.CSSProperties` | - | Inline styles |
| `children` | `React.ReactNode` | - | Provider contents |

### Sidebar

The sidebar container. On mobile it renders as a sheet/drawer; on desktop it supports multiple variants and collapsible behaviors.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `side` | `"left" \| "right"` | `"left"` | Which side of the screen the sidebar appears on |
| `variant` | `"sidebar" \| "floating" \| "inset"` | `"sidebar"` | Visual style variant |
| `collapsible` | `"offExamples" \| "icon" \| "none"` | `"offExamples"` | Collapse behavior: hide, icon-only, or disable collapsing |
| `className` | `string` | - | Additional CSS class names |
| `children` | `React.ReactNode` | - | Sidebar contents |

### SidebarTrigger

Button that toggles the sidebar open/closed state.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `...props` | `React.ComponentProps<typeof Button>` | - | Inherits props from `Button` |

### SidebarRail

Interactive rail for toggling the sidebar on desktop.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `...props` | `React.ComponentProps<"button">` | - | Button props |

### SidebarInset

Main content wrapper that responds to sidebar variant/state (useful for the `inset` layout).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `...props` | `React.ComponentProps<"main">` | - | Main element props |

| `...props` | `React.ComponentProps<typeof Input>` | - | Inherits props from `Input` |

Group label with optional render composition.

Group-level action button with optional render composition.

| Prop | Type | Default | Description |
|------|------|---------|-------------|

| `...props` | `React.ComponentProps<"button">` | - | Button props |

### SidebarMenuButton

Menu item button with variants/sizes, active state, and optional tooltip (shown when collapsed in icon mode).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `isActive` | `boolean` | `false` | Marks the item as active |
| `variant` | `"default" \| "outline"` | `"default"` | Visual variant |
| `size` | `"default" \| "sm" \| "lg"` | `"default"` | Size |
| `tooltip` | `string \| React.ComponentProps<typeof TooltipContent>` | - | Tooltip content/props |
| `render` | `React.ReactElement \| ((props, state) => React.ReactElement)` | - | Replace/compose the rendered element |
| `...props` | `React.ComponentProps<"button">` | - | Button props |

### SidebarMenuAction

Secondary action button for a menu item (optionally only visible on hover).

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `showOnHover` | `boolean` | `false` | Only show the action on hover/focus within |
| `render` | `React.ReactElement \| ((props, state) => React.ReactElement)` | - | Replace/compose the rendered element |
| `...props` | `React.ComponentProps<"button">` | - | Button props |

### SidebarMenuBadge

| Prop | Type | Default | Description |

| Prop | Type | Default | Description |

### SidebarMenuSubButton

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `"sm" \| "md"` | `"md"` | Size |
| `isActive` | `boolean` | `false` | Marks the item as active |
| `render` | `React.ReactElement \| ((props, state) => React.ReactElement)` | - | Replace/compose the rendered element |
| `...props` | `React.ComponentProps<"a">` | - | Anchor props |

### useSidebar

Hook to read and control sidebar state. Must be used within `SidebarProvider`.

| Returns | Type |
|---------|------|
| `state` | `"expanded" \| "collapsed"` |
| `open` | `boolean` |
| `setOpen` | `(open: boolean) => void` |
| `openMobile` | `boolean` |
| `setOpenMobile` | `(open: boolean) => void` |
| `isMobile` | `boolean` |
| `toggleSidebar` | `() => void` |
