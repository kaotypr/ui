## API Reference

This component is built on top of [Base UI Accordion](https://base-ui.com/react/components/accordion).

### Accordion

The root component that wraps all accordion items.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string[]` | - | The controlled value of the item(s) that should be expanded |
| `defaultValue` | `string[]` | - | The uncontrolled value of the item(s) that should be initially expanded |
| `onValueChange` | `(value: string[]) => void` | - | Event handler called when an accordion item is expanded or collapsed |
| `disabled` | `boolean` | `false` | Whether the component should ignore user interaction |
| `multiple` | `boolean` | `false` | Whether multiple items can be open at the same time |
| `orientation` | `"horizontal" \| "vertical"` | `"vertical"` | The visual orientation of the accordion |
| `loopFocus` | `boolean` | `true` | Whether to loop keyboard focus back to the first item when the end of the list is reached |
| `hiddenUntilFound` | `boolean` | `false` | Allows the browser's built-in page search to find and expand the panel contents |
| `keepMounted` | `boolean` | `false` | Whether to keep the element in the DOM while the panel is closed |
| `dir` | `"ltr" \| "rtl"` | - | The reading direction |
| `render` | `ReactElement \| (props, state) => ReactElement` | - | Allows you to replace the component's HTML element with a different tag, or compose it with another component |
| `className` | `string \| (state) => string` | - | CSS class applied to the element, or a function that returns a class based on the component's state |
| `style` | `CSSProperties \| (state) => CSSProperties` | - | Style applied to the element, or a function that returns a style object based on the component's state |

### AccordionItem

Groups an accordion header with the corresponding panel. Must be used within an `Accordion` component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `any` | **Required** | A unique value that identifies this accordion item |
| `disabled` | `boolean` | `false` | Whether the item should ignore user interaction |
| `onOpenChange` | `(open: boolean) => void` | - | Event handler called when the panel is opened or closed |
| `render` | `ReactElement \| (props, state) => ReactElement` | - | Allows you to replace the component's HTML element with a different tag, or compose it with another component |
| `className` | `string \| (state) => string` | - | CSS class applied to the element, or a function that returns a class based on the component's state |
| `style` | `CSSProperties \| (state) => CSSProperties` | - | Style applied to the element, or a function that returns a style object based on the component's state |

### AccordionTrigger

The button that toggles the visibility of the item's content. Must be used within an `AccordionItem` component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The trigger content |
| `nativeButton` | `boolean` | `true` | Whether the component renders a native `<button>` element when replacing it via the `render` prop. Set to `false` if the rendered element is not a button |
| `render` | `ReactElement \| (props, state) => ReactElement` | - | Allows you to replace the component's HTML element with a different tag, or compose it with another component |
| `className` | `string \| (state) => string` | - | CSS class applied to the element, or a function that returns a class based on the component's state |
| `style` | `CSSProperties \| (state) => CSSProperties` | - | Style applied to the element, or a function that returns a style object based on the component's state |

### AccordionContent

The content area that is shown or hidden when the item is toggled. Must be used within an `AccordionItem` component.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The panel content |
| `hiddenUntilFound` | `boolean` | `false` | Allows the browser's built-in page search to find and expand the panel contents |
| `keepMounted` | `boolean` | `false` | Whether to keep the element in the DOM while the panel is closed |
| `render` | `ReactElement \| (props, state) => ReactElement` | - | Allows you to replace the component's HTML element with a different tag, or compose it with another component |
| `className` | `string \| (state) => string` | - | CSS class applied to the element, or a function that returns a class based on the component's state |
| `style` | `CSSProperties \| (state) => CSSProperties` | - | Style applied to the element, or a function that returns a style object based on the component's state |
