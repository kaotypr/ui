## API Reference

The Kbd components are custom UI primitives for displaying keyboard keys and shortcuts. They render semantic `<kbd>` elements with consistent styling.

### Kbd

Renders a single keyboard key indicator. Accepts all standard `<kbd>` HTML attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | The key label to display (e.g. "Ctrl", "Enter", "âŒ˜") |
| `className` | `string` | - | Additional CSS classes |
| ... | `React.ComponentProps<"kbd">` | - | Any other HTML kbd attributes |

### KbdGroup

A container for grouping multiple `Kbd` components and separators (e.g. `+`) into a single shortcut. Renders a flex container. Accepts all standard `<div>` HTML attributes.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | `Kbd` components and separator elements (e.g. `<span>+</span>`) |
| `className` | `string` | - | Additional CSS classes |
| ... | `React.ComponentProps<"div">` | - | Any other HTML div attributes |
