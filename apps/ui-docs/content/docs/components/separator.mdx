## API Reference

This component is built on top of [Base UI Separator](https://base-ui.com/react/components/separator).

### Separator

An accessible separator that supports horizontal and vertical orientations.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | The orientation of the separator. |
| `render` | `ReactElement \| ((props: HTMLProps, state: Separator.State) => ReactElement)` | - | Replace the underlying element or compose it with another component. |
| `className` | `string \| ((state: Separator.State) => string \| undefined)` | - | CSS class, or a function that returns a class based on component state. |
| `style` | `CSSProperties \| ((state: Separator.State) => CSSProperties \| undefined)` | - | Inline styles, or a function that returns styles based on component state. |
