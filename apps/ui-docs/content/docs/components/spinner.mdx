## API Reference

This component renders a Phosphor icon (`SpinnerGapIcon`) from [`@phosphor-icons/react`](https://phosphoricons.com/).

### Spinner

A presentational spinner that renders an animated SVG icon.

It accepts all standard SVG props via `React.ComponentProps<"svg">`. In addition, it applies:

- `role="status"`
- `aria-label="Loading"`
- `className="size-4 animate-spin"`

You can override `className`, `role`, and `aria-label` by passing your own props.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | - | Additional class names to apply (e.g. `size-6 text-muted-foreground`) |
| `role` | `string` | `"status"` | ARIA role attribute |
| `aria-label` | `string` | `"Loading"` | Accessible label announced to assistive technology |
| `...props` | `React.ComponentProps<"svg">` | - | Any additional SVG props (e.g. `width`, `height`, `color`) |
