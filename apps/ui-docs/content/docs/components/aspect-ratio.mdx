## API Reference

The component is a custom implementation that uses a CSS custom property (`--ratio`) with Tailwind's arbitrary value `aspect-(--ratio)` so content (e.g. images, iframes) fills the space while the container keeps the ratio.

### AspectRatio

The root component that maintains a given aspect ratio for its children. Renders a `div` with CSS custom property `--ratio` and uses Tailwind's arbitrary value `aspect-(--ratio)` so content (e.g. images, iframes) can fill the space while the container keeps the ratio.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `ratio` | `number` | **Required** | The aspect ratio to maintain, expressed as width/height (e.g., `16/9`, `4/3`, `1` for square) |
| `className` | `string` | - | Additional CSS class names to apply |
| `children` | `ReactNode` | - | The content to render within the aspect ratio container (e.g. an image with `fill` or an iframe) |

All other standard `div` props are supported (`style`, `id`, `data-*`, etc.).
