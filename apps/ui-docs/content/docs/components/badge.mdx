## API Reference

This component uses [Base UI's useRender](https://base-ui.com/react/utils/use-render) for flexible rendering (e.g. as `<a>` or `<button>`).

### Badge

Renders a small status descriptor. By default renders as a `<span>`.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"` | `"default"` | The visual variant of the badge |
| `render` | `ReactElement \| (props, state) => ReactElement` | - | Customize the rendered element (e.g. `<a href="#">` or `<button>`) |
| `className` | `string` | - | Additional CSS class names |
| `children` | `ReactNode` | - | Content to display inside the badge |

All other props are forwarded to the underlying element (span, or the element from `render`).

### badgeVariants

A `class-variance-authority` (cva) helper. Use it when you need to apply badge styles to a custom element:

```tsx
import { badgeVariants } from "@kaotypr/ui/badge"

<span className={badgeVariants({ variant: "secondary" })}>Custom</span>
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `variant` | `"default" \| "secondary" \| "destructive" \| "outline" \| "ghost" \| "link"` | `"default"` | The visual variant |
