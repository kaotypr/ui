## Import

This pattern uses [Item](/components/item) and [Progress](/components/progress). Import the building blocks:

```tsx
import {
  Item,
  ItemActions,
  ItemContent,
  ItemGroup,
  ItemMedia,
  ItemTitle,
} from "@kaotypr/ui/item"
import { Progress } from "@kaotypr/ui/progress"
```

## API Reference

File upload is a **pattern** composed of existing components, not a single primitive. Use the following:

| Component | Purpose |
|-----------|---------|
| [Item](/components/item) | Row container for each file (name, progress, actions) |
| [ItemGroup](/components/item) | Wraps the list of file rows |
| [ItemMedia](/components/item) | Icon or thumbnail for the file |
| [ItemContent](/components/item) | Holds file name and optional metadata |
| [ItemTitle](/components/item) | File name text |
| [ItemActions](/components/item) | Time remaining or remove button |
| [Progress](/components/progress) | Upload progress bar |

For full prop APIs, see [Item](/components/item) and [Progress](/components/progress).
